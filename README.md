# koinly-exporter
Scripts to generate CSV reports from Subscan API to import into Koinly

## Dependencies
 - python >=3

## Example usage
python3 koinly-exporter.py hydration $YOUR_SUBSCAN_API_KEY $YOUR_ADDRESS output.csv
